# 面试题10：二进制中1的个数

## 题目描述

输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

## 求解策略

把整数减去1，再和原整数做与运算，就会把该整数右边一个1变成0。那么一个整数中有多少1，就可以进行多少次这样的操作。

## 实现代码

```c++
int  NumberOf1(int n) {
    int num = 0;
    while (n){
        ++num;
        n = n & (n-1);
    }
    return num;
}
```
